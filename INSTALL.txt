Installation instructions:

1) Download the whole package of files from http://drupal.org/project/date.

2) Upload the date files to a 'date' folder in the modules directory. The package
   includes files needed by the Date API, and optional modules to to create
   CCK date fields.

3) Go to admin/build/modules and enable the Date API module. 

You should end up with a structure like: 

   /drupal/sites/all/modules/date/date_api.info
   /drupal/sites/all/modules/date/date_api.install
   /drupal/sites/all/modules/date/date_api.module
   /drupal/sites/all/modules/date/date_api.inc
   /drupal/sites/all/modules/date/date_forms.inc
   /drupal/sites/all/modules/date/date_ical.inc

   /drupal/sites/all/modules/date/date/date.info
   /drupal/sites/all/modules/date/date/date.install
   /drupal/sites/all/modules/date/date/date.module

   /drupal/sites/all/modules/date/date_copy/date_copy.info
   /drupal/sites/all/modules/date/date_copy/date_copy.module

   /drupal/sites/all/modules/date/date_php4/date_php4.inc
   /drupal/sites/all/modules/date/date_php4/date_php4_lib.inc
   /drupal/sites/all/modules/date/date_php4/date_php4_calc.inc
   /drupal/sites/all/modules/date/date_php4/date_php4_tz_identifiers.inc
   /drupal/sites/all/modules/date/date_php4/date_php4_tz_abbreviations.inc
   /drupal/sites/all/modules/date/date_php4/date_php4_tz_map.inc


PHP 4 substitutions are made for PHP 5 date handling when needed by enabling the Date PHP4
module. 

CCK Date Field installation instructions:

1) The CCK date field is included in the Date files at http://drupal.org/project/date.

2) Go to admin/build/modules and enable the date module. Be sure that the Date API module
   and the Content module are also installed.

3) Go to admin/content/types to view cck content types and edit a content type.

4) While viewing a content type, select the option to add a new field from the tabs at
   the top of the page. Several options for date fields should be visible.
   
More documentation is available at http://drupal.org/node/92460.
